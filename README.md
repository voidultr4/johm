# Johm Bio Link

A clean cyber-style bio link page for Johm. It is built as a simple static website, so it can run directly on GitHub Pages without any build tools or extra setup.

## Preview

This site includes quick links for:

- Email
- Discord
- TikTok
- Instagram
- Roblox
- Minecraft

Each link opens the matching contact or social profile.

## Files

- `index.html` - the full website
- `563848924_17926281492132582_4505738337428840039_n.jpg` - profile picture
- `README.md` - information about this project
- `.gitignore` - ignores common system files

## How To Use

Open `index.html` in a browser to view the page locally.

To edit the page, open `index.html` and update the names, links, or colors inside the file.

## Publish With GitHub Pages

1. Upload this project to a GitHub repository.
2. Go to the repository's **Settings**.
3. Open **Pages** from the left menu.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select the `main` branch and `/root` folder.
6. Save the settings.

After GitHub finishes publishing, the site will be available from the GitHub Pages link shown in the Pages settings.
